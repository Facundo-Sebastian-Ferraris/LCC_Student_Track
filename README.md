Puedes descargar como xls el plan de estudio del SIU y usarlo para cargar tu progreso. Puedes modificar nodos manualmente si asi lo deseas.

Por el momento solo esta disponible en Linux como .deb para los usuarios con distro hija de Debian, o Debian mismo.
Y o sino, contar con la AppImage que es la version portable.

Para los que descargan el .deb deben utilizar estos dos comandos:
```bash
sudo dpkg -i lcc-student-track_1.0.0_amd64.deb
sudo apt --fix-broken install
```
Y con la AppImage basta solo con abrirlo.
